# page-builder-v2

AI 검색 최적화형 상세페이지/SEO 콘텐츠 생성기 V2

분석 기준: 업로드된 AI최적화 채팅대화 fileciteturn7file0

## 업그레이드 방향
- 기존 V1 상세페이지 출력물의 차례를 완전히 버리기보다,
  AI 검색에 필요한 블록을 별도 V2에서 생성
- 검색 질문 중심 입력
- AI 검색 친화 요약 / FAQ / ALT / 구조화데이터용 정리 추가
- 상세페이지, 블로그, 릴스 확장에 재사용 가능하도록 설계

## 실행
pip install -r requirements.txt
streamlit run app.py
